O que é um repositório no Git e qual a sua importância?
 Um repositório Git tem a finalidade de permitir o usuário salvar seu projeto no GitHub (nuvem),
 e futuramente permitir que ele extraia seu projeto salvo no mesmo, além disso, me permite ver o histórico de modificações no projeto

Qual a função do Git Bash?
 Git Bash (Bourne Again Shell) é um programa que exibe um terminal, ele me da uma forma de usar o git por meio de comandos de texto

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
 O comando git add prepara os arquivos para serem enviados para o repositório "git add ." 
 me adiciona tudo que está presente na pasta, o "git add <arquivo>" me adiciona apenas o arquivo mencionado.

Por que utilizamos o ponto (.) no comando git add .?
 É utilizado para mencionar o diretório presente, (.) no comando git add significa mencionar tudo presente naquela pasta/diretório

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
 O comando git commit serve de certa forma para declarar a mudança feita, como é possível acessar o histórico de mudanças, é importante separar
 bem como e quais mudanças foram feitas.

Qual a função do comando git push?
 O comando git push envia todas as mudanças realizadas no projeto para o repositório remoto

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
 Commits servem de certa forma para localizar mudanças feitas ou erros de forma prática, caso algo de errado podemos voltar
 no ponto antes daquele commit

Explique a função das tags HTML e como elas estruturam uma página.
 As tags são o que definem tudo, como blocos de construção, cada uma tem um significado e entre elas existem hierarquias, 
 são palavras chaves que se encaixam entre as <> sempre em pares, por ex: <h1></h1>

O que são atributos em uma tag HTML e qual a sua função?
 Os atributos de uma tag HTML servem como informações extras que moldam a tag, é como você "personaliza" ela

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
 Emmet é um tipo de abreviação, eu digito apenas div*8 e ele cria oito chaves <div></div> para mim, ele completa os códigos,
  "!" cria a estrutura básica do html

O que é o box-sizing no CSS e por que usamos ele no reset inicial?
 Box-sizing tem como função definir a largura e altura de um elemento
 usamos o reset inicial no css como forma de remover os padrões criados pelos navegadores, que acabam "desencaixando" os itens criados
 

Por que é importante utilizar tags semânticas no HTML?
 São basicamente os rótulos, elas nos ajudam a localizar facilmente o código na tela, o que ajuda muito na manutenção
 Ao invés de usar apenas <div> eu uso <article>, <nav> e <aside> 

O que são media queries no CSS e qual a sua função?
 Media queries no css tem como função criar regras/padrões pra certos tipos de tela a depender do dispositivo,
  como largura da tela, altura e orientação, tornando o layout responsivo


Qual a função do GitHub Pages?
 GitHub Pages é uma função que me permite visualizar meu projeto no navegador de forma gratuita, não só o meu, como também o de qualquer 
 projeto que possua o GitHub Pages ativado

O que é o Flexbox no CSS e para que ele é utilizado?
 O Flexbox é uma ferramenta do CSS que me permite a organizar os elementos de uma página da web de forma flexível e responsiva
 ele me permite criar uma interface responsiva independente do tamanho da tela

Qual a diferença entre justify-content e align-items em um container Flexbox?
 Em um container Flexbox, justify-content alinha itens pelo eixo horizontal, já 
 o align-items alinha no eixo vertical


Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
 Flex-grow define como um item flexível cresce/se alinha dentro de um container Flexbox,
 ele permite que você controle o espaço entre os itens

Explique a diferença entre flex-direction: column; e flex-direction: row;.
 Flex-direction: column; irá definir que o container flexbox seja partido em colunas, um abaixo do outro
 Flex-direction: row; define que o container flexbox seja partido em uma fileira horizontal, um do lado do outro
